# 🌐 Personal Portfolio Website

## 📌 Project Overview

This project is a **personal portfolio website** designed to showcase skills, projects, certifications, and academic background. It is built using **HTML and CSS**, focusing on a modern UI/UX with responsive design and smooth animations.

The portfolio highlights:

* Personal introduction and career objective
* Technical skills across programming, data science, and AI
* Featured projects with descriptions and GitHub links
* Certifications and academic achievements
* Contact information for professional networking

The design emphasizes **clean aesthetics, interactive elements, and accessibility across devices**.

---

## ⚙️ Setup Instructions

Follow these steps to set up the project locally:

1. **Download or Clone the Repository**

   ```bash
   git clone <your-repo-link>
   ```

2. **Project Structure**

   ```
   ├── index.html
   ├── style.css
   ├── profile.jpeg
   ├── kasi_resume.pdf
   ```

3. **Ensure all files are in the same directory**

   * `index.html` → main webpage
   * `style.css` → styling and layout
   * `profile.jpeg` → profile image
   * `kasi_resume.pdf` → downloadable resume

4. **No additional dependencies required**

   * This is a static frontend project (no frameworks or backend setup needed)

---

## ▶️ How to Run the Project

### Method 1: Direct Open

* Simply double-click `index.html`
* It will open in your default browser

### Method 2: Using Live Server (Recommended)

If you're using VS Code:

1. Install **Live Server Extension**
2. Right-click on `index.html`
3. Click **"Open with Live Server"**

This provides:

* Auto-refresh on changes
* Better development experience
